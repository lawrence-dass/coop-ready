# CI/CD Workflows

See main guide: [CLAUDE.md](../CLAUDE.md)

---

## GitHub Actions Workflows

**`.github/workflows/test-suite.yml`**
- Full test suite (unit + e2e)
- Triggers: Push to main, PRs
- Artifacts: Playwright reports, test artifacts

**`.github/workflows/e2e-tests.yml`**
- E2E tests only
- Triggers: Push to main, PRs

---

## Test Artifacts

- **Playwright reports:** `playwright-report/`
- **Test artifacts:** `playwright-artifacts/` (screenshots, videos, traces)
- **Coverage:** Generated by Vitest

---

## Workflow Triggers

| Trigger | Workflow | Tests Run |
|---------|----------|-----------|
| Push to main | test-suite.yml | All |
| Pull Request | test-suite.yml | All (@P0 critical) |
| Scheduled (nightly) | test-suite.yml | All (optional @P2) |
